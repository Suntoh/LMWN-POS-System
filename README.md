# POS-System

A Point of Sale (POS) system.

## Features

- Track sales reports
- Process sales transactions
- Add new orders
- Pay order
- Generate receipts

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/LMWN/POS-System.git

# Navigate to the project directory
cd POS-System

# Install dependencies
npm install
```

### Running the Application

```bash
# Development mode
npm run dev

# Production build
npm run build
npm start
```
